## Development
  You can choose one of three package managers: `npm`, `yarn`, `pnpm`.

<details>
  <summary><code>npm</code></summary>

  <h5>Installation</h5>
  <pre><code>
  npm install
  </code></pre>
  <h5>Run</h5>
  <pre><code>
  npm run dev
  </code></pre>
  <h5>Build</h5>
  <pre><code>
  npm run build
  </code></pre>
</details>

<details>
  <summary><code>yarn</code></summary>

  <h5>Installation</h5>
  <pre><code>
  yarn install
  </code></pre>
  <h5>Run</h5>
  <pre><code>
  yarn run dev
  </code></pre>
  <h5>Build</h5>
  <pre><code>
  yarn run build
  </code></pre>
</details>

<details>
  <summary><code>pnpm</code></summary>

  <h5>Installation</h5>
  <pre><code>
  pnpm install
  </code></pre>
  <h5>Run</h5>
  <pre><code>
  pnpm run dev
  </code></pre>
  <h5>Build</h5>
  <pre><code>
  pnpm run build
  </code></pre>
</details>
